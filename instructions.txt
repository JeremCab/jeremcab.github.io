Adding content on my website:

1. Add the file (paper, presentation, etc.) to:

files/papers/         or
Files/presentations/

2. Update the page in:

_pages/my_papers.md         or
_pages/my_presentations.md

3. Using command line, go to my repo:

jeremcab.github.io

add the content, commit and push.

git add .
git commit -m <message>
git push

The webpage is then be updated and deployed automatically.
